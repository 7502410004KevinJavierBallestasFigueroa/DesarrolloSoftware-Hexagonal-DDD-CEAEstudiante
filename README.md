# DesarrolloSoftware-Hexagonal-DDD-CEAEstudiante

Es una API en Spring Boot 3 con Java 21 que aplica Arquitectura Hexagonal y DDD, usando PostgreSQL, Flyway para migraciones, JWT para autenticación y Swagger para documentar. Expone endpoints para autenticación y gestión de usuarios, con un usuario admin creado por defecto y toda la configuración lista vía application.yml y Postman.
